Explore your GraphQL schema and make live calls against your Uni.

## Setup

Install dependencies:

```
npm install
# or
yarn install
```

Start the server:

```
npm run start
# or
yarn start
```

Your browser will automatically open to http://localhost:3000 with the explorer open.
